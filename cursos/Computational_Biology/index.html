MATE 8990 Topics in Applied Mathematics: Computational Cell Biology

Course syllabus

Topics

  • Introduction to mathematical modeling
  • Introduction to ordinary differential equations
  • Voltage gated ionic currents
  • Carrier proteins and co-transporters
  • Na+ transport in a whole cell
  • Parameter computation

Final project

Possible topics:
  • A model of an erythrocyte cell (red blood cell), which includes principal ion transport and volume regulation.
  • A model of a neuron, which includes principal ion transport and volume regulation.
  • A Bayesian framework for doing parameter estimation in a transport model, e.g., the glut model. The framework can be based on the work published by Bayarri et al., you can download a working paper file from this site.
  • A parallel implementation of the whole cell model.
The project will be presented to the class in an oral presentation of 1/2 hour on December 3 and 5, two presentations per day. A project report, which must incorporate any comment during the presentation, will be uploaded to Moodle on Thursday, December 20.
